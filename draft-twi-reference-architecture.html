Trusted Workload Identity Reference Architecture

Abstract

Technological advancement coupled with security concerns demands more and more use cases, that require data-in-use protections. In addition regulatory bodies around the world increasingly require data-in-use protection and privacy enhancing technologies. As these advanced technologies are been deployed, challenge remains how to identify the workloads and put trust behind them? This draft will address the core issue of managing a trustworthy workload identity meeting Confidential Computing requirements.¶
